# .codex/PR-21.md — Conflict Marking & Superseding Semantics (Production Rules)

## Goal
Implement production-grade conflict handling: supersedes links, contradiction flags, and trust-aware conflict marking.

## Deliverables
- Define standard link semantics:
  - `supersedes`, `references`, `confirms`, `contradicts`
- On ingestion / remember:
  - if an atom supersedes another, mark old atom as superseded (flag)
  - if contradict links appear, add contradiction metadata for ranking
- Update ranking rules to:
  - down-rank contradicted atoms
  - prefer confirmed atoms
  - respect trust ordering

## Tests
- Supersede chain produces expected “current truth” ordering.
- Contradicted atoms are down-ranked but still retrievable.

## Acceptance
- ask() results reflect supersede/contradiction rules deterministically.
