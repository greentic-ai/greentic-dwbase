## Goal

## Deliverables
- Define quarantine world convention:
  - `quarantine:<original_world>`
- Add anomaly triggers:
  - spam rate spikes

- When triggered:

  - add `suspect` flag
- Update ask() default behavior:
  - exclude quarantine worlds unless explicitly requested

## Tests
- Suspicious atom ends up quarantined.
- Normal ask excludes quarantined by default.

## Acceptance
- Quarantine is observable and auditable, not silent deletion.
