## Goal
Provide operational backup/restore for production deployments.

## Deliverables
- Snapshot format:
  - tar/zstd of log segments + minimal metadata + index metadata
- CLI:
  - `dwbase backup create --out snapshot.tzst`
  - `dwbase backup restore --in snapshot.tzst --data-dir ...`
- Ensure restore supports:
  - clean startup without rebuild (optional)
  - rebuild if index missing

## Tests
- Create snapshot, wipe data dir, restore, verify atoms exist and ask() works.

## Acceptance
- Backup/restore documented and runnable.
